Directivas para mostrar y ocultar tabs


Uso de directiva tipo boton o link actionTab para mostrar o ocultar los tabs llamando en base a atributos.
Esta directiva siempre dependera de las directivas: ngTabs y ngPane respectivamente.


  
    <action-tab icon="" title="Mostrar Tab2" action="hide" type="button" numtab="1"></action-tab>      
    <action-tab icon="" title="Mostrar Tab2" action="show" type="link" numtab="2"></action-tab>
    <action-tab icon="" title="Mostrar Tab3" action="show" type="button" numtab="3"></action-tab>

Especificaciones


Atributo Descripci√≥n
show-all Con el valor en TRUE se puede mostrar todas las tabs activas / <ng-wizard show-all="TRUE">
icon Permite definir el icono a mostrarse a tra ves de una clase, ejemplo icon="icon-mail-forward", solo configurar en caso de usar type="link"
type Define si la directiva se comportara como boton o link segun se ponga button o link
title Define el texto a mostrarse
action Permite ocultar o mostrar un tab
numtab Define el numero de tab a ocultar o mostrar
