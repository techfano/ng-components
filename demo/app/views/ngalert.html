ngAlert




Inicializar el componente ngAlert de la siguiente manera:

en HTML

  1. <ui-alert stoptions= "alert">
  2. </ui-alert>

en Javascript

  1. $scope.alert={
  2.   message:'Not Found',
  3.   type:'info',
  4.   show:true,
  5.   hide:{auto:false,click:true,time:0}
  6. }

Despues de inicializar el objeto alert en javascript, se puede seguir modificando la directiva en diferentes eventos

  
  1. $scope.warning=function(){
  2.   $scope.alert.type="warning";
  3. }
  4. $scope.error=function(){
  5.   $scope.alert.type="error";
  6. }
  7. $scope.new=function(){
  8.   $scope.alert.message="New Message";
  9. }
  10. $scope.time=function(){
  11.   $scope.alert.hide.auto=true;
  12.   $scope.alert.hide.time=5000;
  13. }

Especificaciones


Propiedad Tipo Descripci√≥n
message Caracter Imprime el mensaje dentro de la alerta
type Caracter Define el tipo de alerta ('info','success','warning','error')
message Caracter Imprime el mensaje dentro de la alerta.
hide objeto Utiliza tres propiedades para definir como ocultar la alerta.
hide.auto booleano(true,false) permite habilitar el cierre de la directiva automaticamente.
hide.time Entero Define el tiempo en el que se ocultara la alerta, si esta propiedad no es definida, se ocultara automaticamente en 5 segundos.
hide.click booleano(true,false) Oculta o muestra el boton de cierre de la alerta.
